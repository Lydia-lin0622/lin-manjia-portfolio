📌 如何植入视频
B站：打开视频 → 点击"分享" → "嵌入代码" → 复制 <iframe> → 粘贴到文件中替换 <div class="video-ph">...</div>
YouTube：视频下方 → "分享" → "嵌入" → 复制代码 → 同上替换
本地 MP4：将 mp4 文件放在同文件夹，替换为 <video controls src="文件名.mp4" style="width:100%;height:100%;position:absolute;top:0;left:0;"></video>
抖音：暂不支持嵌入，建议截图放在作品卡片区。
